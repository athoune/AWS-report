AWS report reader
=================

Graph from AWS's CSV report. It produces a HTML page with flot graph.

Install
-------

No install, no dependencies. Fetch the source.

Usage
-----

For now it handles :

 * SDB
 * SQS
 * S3
 * CloudFront
 * SES

Command
-------

		./report.py report.csv
